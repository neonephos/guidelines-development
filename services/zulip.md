# Zulip

Zulip is the preferred chat service for NeoNephos Foundation projects.

Zulip channels can be requested in the [Project IT Services Support Portal](https://jira.linuxfoundation.org/plugins/servlet/desk/portal/2/create/1506).

## Default Setup

Each project can request the following channels:

- **`neonephos-{project}-discussion`** — Public channel for project discussions.
- **`neonephos-{project}-support`** — Public channel for project-related support requests.
- **`neonephos-{project}-tsc`** — Public channel for TSC communication.
- **`neonephos-{project}-tsc-private`** — Private channel for TSC communication.
